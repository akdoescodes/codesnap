# **App Name**: CodeSnap

## Core Features:

- HTML Input: Takes HTML code as input, either by pasting or uploading a file.
- Dimension Input: Allows users to specify the dimensions (width and height) for the screenshot.
- HTML Rendering: Renders the HTML code in an iframe, applying Bootstrap styles.
- Screenshot Capture: Uses html2canvas to capture a screenshot of the rendered HTML at the specified dimensions.
- Screenshot Download: Provides a download link for the captured screenshot (PNG format).

## Style Guidelines:

- Primary color: #2ecc71 (Green) for a fresh and modern look.
- Secondary color: #f0f0f0 (Light Gray) for backgrounds and subtle contrasts.
- Accent color: #f39c12 (Orange) for buttons and interactive elements.
- Clean and modern typography to ensure readability of code snippets and instructions.
- Simple and intuitive layout with clear sections for input, preview, and download.
- Subtle loading animations while rendering and capturing the screenshot.
